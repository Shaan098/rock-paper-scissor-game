# 🎮 Rock Paper Scissors Game

## 🧾 Project Description

The **Rock Paper Scissors Game** is a fun and interactive web project built entirely using **JavaScript**, **HTML**, and **CSS**.  
It replicates the classic hand game where a player competes against the computer by choosing between **Rock**, **Paper**, or **Scissors**.  
The computer’s choice is generated randomly, and the result is displayed instantly on the screen.

The main objective of this project is to demonstrate how JavaScript can be used to add interactivity and dynamic behavior to a webpage without relying on any external libraries or frameworks.  
It focuses purely on **core JavaScript logic**, **DOM manipulation**, and **event handling**.

---

## ⚙️ Features

- 🪨 **Click-based gameplay:** Choose Rock, Paper, or Scissors through interactive image buttons.  
- 🧠 **Computer logic:** Computer generates its choice randomly using `Math.random()`.  
- 📊 **Instant results:** Displays both the player’s and the computer’s choices along with the winner.  
- 💻 **Clean interface:** Simple and visually appealing design using CSS styling.  
- 🔁 **Pure JavaScript:** No frameworks or libraries used — completely hand-coded logic.  
- 📱 **Responsive design:** Works on both desktop and mobile browsers.

---

## 🧠 Game Rules

- **Rock** beats **Scissors**  
- **Scissors** beats **Paper**  
- **Paper** beats **Rock**  
- If both choose the same option → it’s a **Tie**

---

## 🧩 Technologies Used

- **HTML5** – Structure and layout  
- **CSS3** – Styling, colors, and animations  
- **JavaScript (ES6)** – Game logic, randomization, and DOM updates  

---

## 🚀 How to Run the Project

1. Clone or download this repository.  
2. Open the folder and locate the file named **`index.html`**.  
3. Double-click to open it in any modern browser (Chrome, Edge, Firefox, etc.).  
4. Click on **Rock**, **Paper**, or **Scissors** image to play.
